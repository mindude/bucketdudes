1. Create a web app to be able to send message to a specific Slack channel. The web app will have:
• HTML View + angular controller to call your own end point.
• API Gateway end point available to the world with an application key (you will generate this app key in AWS Console)
• Lambda integrated with API Gateway end point that will log the action and post the message to the Slack channel.
• Integration with Github, so every time a push is made to the repo it will post a message to the team group.
• 100 points.
